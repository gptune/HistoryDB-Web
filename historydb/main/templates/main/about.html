GPTune History Database


GPTune is an autotuning framework which is designed to tune high-performance application codes as "black-boxes", running them for carefully chosen tuning parameter values and building a performance model based on the measured performance. GPTune has a nice feature called History Database which allows users to save and re-use their performance data to reduce the cost of the expensive black-box objective function (i.e. run application and measure the performance on a parallel machine).

Harness the Power of Crowdtuning


The history database can leverage a shared public database provided by this web application, where users can store their performance data or download the performance data provided by other users. With the shared database, everyone can benefit from (expensive) runs of widely used high-performance computing codes. This website provides a nice dashboard to search and download performance data from the public repository and an online user manual to learn how to use the database.

Join our Community


Unregistered users can only view publicly available performance data. This means that they cannot view data disclosed to authorized users or upload any kind of data into out database.

Please Sign Up to access more data in our history database! Registered users can download data disclosed to registered users. There is another privilege level called Certified for registered users. The GPTune team will update the user status as "certified" if the user has entered user profile information correctly. Certified users can access data for certified users and create collaboration groups to share data within your group members.
