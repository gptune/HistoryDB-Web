GPTune History Database


GPTune is an autotuning framework which is designed to tune high-performance application codes as "black-boxes", running them for carefully chosen tuning parameter values and building a performance model based on the measured performance. GPTune History Database allows users to save and re-use their performance data to reduce the cost of the expensive black-box objective function (i.e. run application and measure the performance on a parallel machine).

Harness the Power of Crowdtuning


The history database can leverage a shared public database provided by this webpage, where users can store their performance data or download the performance data provided by other users. With the shared database, everyone can benefit from (expensive) runs of widely used high-performance computing codes. This website provides a nice dashboard to search and download performance data from the public repository and an online user manual to learn how to use the database.

GPTune is a joint research project between Lawrence Berkeley National Laboratory and University of California at Berkeley, and is part of the xSDK4ECP effort supported by the Exascale Computing Project (ECP).
