GPTune History Database


GPTune is an autotuning framework that relies on multitask and transfer learnings to help solve the underlying black-box optimization problem. GPTune is part of the xSDK4ECP effort supported by the Exascale Computing Project (ECP).

GPTune provides a history database for GPTune, where all users will have the option to store and re-use their performance data, to allow GPTune to achieve the best possible performance tuning.

To harness the power of crowdtuning, this webpage provides a shared public database, where users can store their performance data or download the performance data provided by other users. With the shared database, everyone can benefit from (expensive)runs of widely used high-performance computing codes.
