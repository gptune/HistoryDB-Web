GPTune History Database


GPTune is an autotuning framework that relies on multitask and transfer learnings to help solve the underlying black-box optimization problem. GPTune is part of the xSDK4ECP effort supported by the Exascale Computing Project (ECP).

GPTune provides a history database which allows users to save and re-use their performance data (e.g. function evaluation data, trained surrogate models) to relieve the issue of the expensive black-box objective function (i.e. run application and measure the performance on a parallel machine).

To harness the power of crowdtuning, the GPTune history database can leverage a shared public database provided by this webpage, where users can store their performance data or download the performance data provided by other users. With the shared database, everyone can benefit from (expensive) runs of widely used high-performance computing codes.
