GPTune History Database

GPTune is an autotuner for high-performance computing codes, relying on multitask learning to help solve the underlying black-box optimization problem. GPTune is part of the xSDK4ECP project supported by the Exascale Computing Project (ECP). The GPTune development team (GPTune-Dev) is developing a \textit{history database} for GPTune, where all GPTune users will have the option to store their performance data, or download and use the performance data from other users, to allow GPTune to achieve the best possible performance tuning for widely used codes. In this document, we describe the current design of the GPTune history database and explain how to use it.
